# LinkedIn — OSS launch post

Alyra Labs is now open source (MIT).

It's a virtual chemistry desk: pour reactants on wood, watch a real equation resolve — `HCl + NaOH → NaCl + H2O` — and get a plain-language tutor explanation. Perfume atelier, XP, OCR notes, and teacher class codes are in the same app.

Try it: https://alyra-labs.vercel.app  
Code: https://github.com/Ghost-ops721/alyra-labs

If you teach or learn chemistry — or just like tactile product UI — I'd value a try, a PR, or a README review.

If Alyra Labs helped you, consider giving it a ⭐ so more educators and developers can discover it.
