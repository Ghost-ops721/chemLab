**Title:** Show HN: Alyra Labs – open-source virtual chemistry lab (MIT)

**URL:** https://alyra-labs.vercel.app

---

Alyra Labs is an open-source virtual chemistry laboratory. You drag glassware onto a desk, pour reactants, and watch a real balanced equation resolve — e.g. `HCl + NaOH → NaCl + H2O` — then get a plain-language tutor explanation.

Also: perfume atelier (top/heart/base notes), XP/badges, OCR note scan, teacher class codes.

- Live: https://alyra-labs.vercel.app
- Source (MIT): https://github.com/Ghost-ops721/alyra-labs
- Desk route: /lab
- Good first issues: labeled on the repo

Built with Next.js, a reaction engine, and Groq for the tutor. Guests can try two chemicals before signup. Feedback and PRs welcome.

If Alyra Labs helped you, consider giving it a ⭐ so more educators and developers can discover it.
