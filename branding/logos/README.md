# Chem Lab — Logo Concepts

Five concepts, one coherent family (ink `#14241f`, walnut `#4a3428`, seafoam teal `#1a6b5c`/`#8fc0b5`, amber `#c4783a`, foam `#eaf4ef`). All flat vector, no gradients/bevels/photorealism, no purple, no navy-blue corporate-badge clip-art.

| File | Concept | Best use |
|---|---|---|
| `chemlab-logo-01-wordmark.png` | Pure serif wordmark, single teal bubble accent over the "i" | Nav bar, header, anywhere the icon would be redundant |
| `chemlab-logo-02-monogram.png` | "C"+"L" fused into a geometric flask silhouette, walnut fill | App icon, favicon, social avatar (works small) |
| `chemlab-logo-03-icon-lockup.png` | Bespoke flask-with-liquid icon + wordmark side by side | Primary lockup — landing page header, video title cards, deck covers |
| `chemlab-logo-04-abstract-mark.png` | Abstract meniscus-curve mark with rising bubbles | Loading states, app-icon alternate, favicon at tiny sizes, watermark |
| `chemlab-logo-05-badge-seal.png` | Circular badge/seal, "POUR · STIR · REACT" | Merch, achievement/badge system (ties into in-app XP badges), stickers |

**Recommendation:** lead with `03-icon-lockup` as the primary logo, `02-monogram` as the app icon/favicon, and `04-abstract-mark` as a minimal-motion mark for loading/splash states.
