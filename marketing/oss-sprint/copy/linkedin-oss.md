# LinkedIn — OSS launch post

Chem Lab is now open source (MIT).

It's a virtual chemistry desk: pour reactants on a walnut desk, watch a real equation resolve — `HCl + NaOH → NaCl + H2O` — and get a plain-language tutor explanation. Perfume atelier, XP, OCR notes, and teacher class codes are in the same app.

Try it: https://chemistry-nu-five.vercel.app  
Code: https://github.com/Ghost-ops721/chemLab

If you teach or learn chemistry — or just like tactile product UI — I'd value a try, a star, or a PR.
